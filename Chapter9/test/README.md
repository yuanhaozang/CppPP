## C++类中的成员函数和成员变量
*   在对象创建时，成员变量时保存在对象中的，而成员函数是保存在公共代码区

*   这就存在一个问题：Test类中有add一个成员函数，但是它们的函数体中并没有关于不同对象的区分，那么A调用这个函数的时，该如何知道把A设为对象，而并非B呢？

*   其实类中的成员函数都有一个隐藏的指针形参，即this指针。当对象调用成员函数的时候，会隐形的将对象地址作为实参传递给this形参，也就是说this指针会指向调用该函数的对象。因此在函数体中有关成员变量的操作，都是通过该this指针去访问的

*   this指针是成员函数中第一个隐含的指针形参，一般情况下由编译器通过寄存器自动传递

*   this 指针本质上是成员函数的形参，所以其不存储在对象中。this指针一般情况下存在栈里，有些编译器会把它放到寄存器中
*   this 指针变量被const修饰，其指向的值是不能被修改的。this指针只能存在于成员函数内部使用。**this指针不存在于静态成员函数中(静态成员函数如同静态成员变量，它不属于哪一个对象，而是属于整个类的，而this指针却指向一个具体对象)**
```C++

    public:
        int m_year;
        int m_month;
    public:
        void Init(int a, int b)
        {
            m_year  = a;

        };
        /*  
        int add(Test *const this, int a, int b)
        {
            this->m_year  = a;
            this->m_month = b;
        }
        */
};

int main()
{
    /*
        
    */
    Test A;
    A.add(1,3); //A.add(&A, 1, 3);

    Test B;

}
```
