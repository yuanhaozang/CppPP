## 内存分配方式
内存分为五个区:堆、栈、自由存储区、全局\静态存储区和常量存储区
内存分为五个区:堆、栈、代码区、全局\静态存储区和字符串常量区
*  堆
    *   用于程序的内存动态分配，malloc/free

*   栈
    *   局部变量的存储单元，在函数执行结束后这些存储单元自动释放。

*   自由存储区
    *   new/delete。自由存储区可以是堆，也可以是静态存储区，这要看operator new在哪里为对象分配内存

*   静态存储区
    *   静态存储区不区分初始化和未初始化的数据区，因为静态存储区内的变量若不显式初始化，编译器会自动以默认方式初始化
    *   常量区(静态常量，字符串常量，常变量)
    *   全局区(全局变量)
    *   静态区(静态变量)


*   代码区
    *   存放程序体的二进制代码。例如：函数

内存分区可以分为两个:代码区，数据区(堆，栈，静态存储区)

```C++
int a = 0;  //全局区
int main()
{
    int b;  //栈
    char s[] = "abc";   //栈
    char *p1 = "abc";   //"abc"字符串常量区，p1在栈区
    static int c = 0;   //c在静态变量区，0是文字常量，在代码区
    const int d =0;     //栈
    static const int q;  //静态常量区

    p1 = (char *) malloc(10);   //分配十个字节在堆区
    strcpy(p1,"abc");   //"abc"字符串常量区，编译器可能会将它与p1所指向的"abc"优化为一个地方
    return 0;
}
```
